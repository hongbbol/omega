# 고양이 오메가3 평가 프로젝트 — Claude Code 핸드오프

이 폴더만 있으면 작업을 그대로 이어갈 수 있습니다. **이 README를 먼저 읽어주세요.**

## 1. 프로젝트 개요
근거 기반으로 "고양이에게 오메가3가 효과 있는지 / 어떤 제품이 좋은지"를 정리하는 프로젝트.
최종 산출물은 두 가지: **(1) 인스타 콘텐츠**, **(2) 점수화 순위 + 추천**(웹서비스 점수 로직의 토대).
핵심 원칙: 과장 없이, 근거(논문·학회)와 공개 스펙에 기반해서만 평가. **추정치 입력 금지(모르면 빈칸/미상).**

## 2. 파일 구성
- `products.csv` — **제품 데이터(이 파일만 편집).** 33개 제품 × 스펙 컬럼.
- `build_xlsx.py` — 채점 엔진. CSV를 읽어 점수·순위가 든 xlsx를 생성.
- `고양이_오메가3_제품_채점표.xlsx` — 현재 산출물(기준 시트 + 점수표 시트).
- `근거매트릭스_평가체계.md` — 1·2단계 산출물: 효능 근거(적응증 A/B/C)와 채점 루브릭 정의.

## 3. 채점 루브릭 (요약 — 자세한 정의는 근거매트릭스 md)
점수 두 축:
- **축1 품질 점수(0~100, 순위용)** — 가중치: 성분 35 / 산패 30 / 원료·형태 20 / 투명성 15
- **축2 적합성(순위 아님, 매칭 태그)** — 관절·신장 / 일반유지 / 피부보조 등

세부 배점(이 토큰만 써야 점수가 매겨짐 — `build_xlsx.py`의 `LOOKUPS` 참조):
- 성분 35 = 밀도(EPA+DHA mg/단위: <100→5/100-199→10/200-299→15/≥300→20) + 순도(%: <40→3/40-59→6/≥60→10) + 비율표기(명시5/일부2/미표기·미상0)
- 산패 30 = 항산화제(있음10/없음·미상0) + 포장(우수10/보통5/취약·미상0) + 제형(캡슐·소프트젤10/츄·스틱·펌프액상6/대용량액상2/미상0)
- 원료·형태 20 = 형태(rTG·TG10/EE5/미상5) + 어종(소형어10/조류8/일반어6/식물성0/미상3)
- 투명성 15 = 인증(있음5) + COA(공개6/일부3) + 원료사(명시4/일부2)

**안전 점수상한(관문):** 비해양성=ALA단독→총점 20 / 대구간유→40 / 산패고위험→50.
- 단, **조류(algae) 오일은 EPA/DHA를 직접 공급 = 정상 처리(상한 X).** ALA(아마씨·들기름)만 상한.
- **폴락(명태) 어체유는 간유 아님 → 대구간유 상한 대상 아님.**

**빈칸/미상:** 공개 안 한 항목은 해당 세부점수 0(제외 아님). 비공개=점수 자연 감점.
`정보검증` 태그(공개·검증 / 라벨만 / 미상)로 점수의 신뢰도를 별도 표기.
모든 점수는 "공개정보 기반·실측 전"임을 전제(실제 함량·산패·중금속 실측은 다음 단계).

## 5. 작업 방법 (Claude Code)
1. **새 제품 스펙 채우기:** 제품 상세페이지의 **Supplement Facts(1캡슐당 EPA/DHA mg)**, **IFOS/COA 표(산가·과산화물·TOTOX·중금속)**, 원료/제형/포장 정보를 보고 `products.csv` 해당 행을 채운다. (토큰은 위 스키마대로.)
2. **재생성:** `pip install openpyxl` → `python build_xlsx.py` → xlsx 생성.
3. **확인:** xlsx를 Excel/LibreOffice로 열면 수식이 자동 계산(점수·순위·데이터충실도).
4. **루브릭 수정:** `build_xlsx.py`의 `LOOKUPS`(카테고리 배점)·밀도/순도 구간·안전상한 부분을 고친다.

## 6. 데이터 수집 팁
- **네이버 스마트스토어/브랜드몰은 자동 접근 차단** → 못 긁음. 상세페이지 **캡처 이미지**로 받아야 함.
- **자사몰(예: choandkang.com)은 fetch 가능**하나 핵심 수치는 이미지라 텍스트로는 주변 정보만 잡힘 → 결국 Supplement Facts/COA 이미지가 필요.
- 가장 중요한 한 장: **Supplement Facts(1캡슐당 EPA/DHA mg)**. 그 다음 **IFOS Product Report(COA)**.
- IFOS 완제품 5★ 제품은 nutrasource/IFOS에서 배치 실측 COA 확인 가능(라벨값 vs 실측 비교 = 콘텐츠 소재).

## 7. 현재 상태 (스냅샷)
- 33개 중 **완전 입력 4개**, 밀도 확보 9개.
- 잠정 상위: 묘견서 100 · 닥터레이 95 · 조공 레드 90 · 조공 데일리 85 (모두 KD/Solutex rTG·IFOS 완제품).
  - 차이: 밀도(묘견서 320 > 닥터레이 200 > 조공 110) + 포장(레드 블리스터 vs 데일리 병 = 5점) + EPA비율/아스타잔틴.
- 나머지 ~24개(특히 국내 펫·클리닉: 조공 rTG퓨어·펫생각·이퀄·88오메가가드·듀오메가·아이슬란딕헤링·바이오펫츠·살모펫·페스룸·룰루파마·씽크라이크펫·klnix·venefit·오메가플러스펫·뉴트리벳·닥터뮨·아베텍 등)은 밀도가 이미지-잠금 → 상세페이지 캡처로 채워야 함.
- 식물성 처리: 고려은단·뉴트리코어=조류(정상), 닥터뮨=ALA/조류 미확인(상한 보류). 뉴트리벳=폴락 어체유(대구간유 아님, 상한 해제).

## 8. 남은 작업 / 다음 단계
1. 남은 제품 스펙 수집(캡처 → CSV) → 점수·순위 완성.
2. 적합성 태그 정리(관절·신장 / 일반유지 / 피부) → 갈래별 추천 구성.
3. 인스타 콘텐츠: ① "고양이 오메가3, 정말 필요한가"(적응증 A/B/C, 근거매트릭스 md) ② "고르는 기준 6"(품질 4축) ③ "라벨 vs 실측"(산패·표시정확성) ④ 출처 슬라이드.
4. 실측 테스트 단계(라벨로 안 보이는 실제 함량·산패·중금속) — "공개정보 점수"의 다음 단계.

## 9. 근거 요약(콘텐츠용 핵심) — 자세히는 근거매트릭스 md
- 대전제: 고양이는 ALA→EPA/DHA 전환 거의 불가 → 해양성/조류 EPA/DHA만 유효.
- 효능: A(근거강함) 관절염·만성신장병 / B(보조) 피부·심장(심장은 학회권고+개 근거, 고양이 직접근거 약함) / C(불필요) IBD·요로(FIC)·노령인지·건강성묘 유지.
- 4kg 기준 환산: 유지 ≈120~200mg/일, 관절·신장 치료 ≈400~450mg/일(사료 함량 차감).
- 안전: 고양이 안전상한 미설정, 대구간유 회피(비타민A/D), 산패된 오메가3는 유해.
